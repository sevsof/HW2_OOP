## Домашнее задание по технологиям программирования:

В рамках данного домашнего задания я реализовала 4 класса с собственными методами:

1. Ingredient - класс, который хранит название, вес и единицу измерения об ингредиенте. В нем реализовано 4 метода, которые далее использованы в других классах

2. Recipe - класс, который хранит название и список ингредиентов для определенного рецепта. В нем реализовано 6 методов, один из которых является статическим.

3. ShoppingList - класс, который формирует список покупок. В нем реализовано 5 методов.

4. DietaryRecipe - класс наследник от Recipe, который хранит инфмормацию о диетической категории. В данном классе реализовано два переопределенных метода.

Также, в рамках задание выполнены тесты к классам Ingredient, Recipe, ShoppingList (исходя из условия задания)

Все решения по ООП и тестированию разрабатывались в Git-репозитории и опубликованы на GitHub.

Добавлен корректный для Python `.gitignore`, чтобы в репозиторий не попадали окружение, кэши и файлы IDE.

Также, в данный момент создан и написан файл README

## Использование:

### Для установки:

```bash
   git clone https://github.com/sevsof/HW2_OOP
   cd HW2_OOP
   pip install -r requirements.txt
```
### Для запуска код:

```bash
python classes.py
```

### Для запуска тестов:

```bash
pytest test.py
```
